# Awareness vs Thinking (Operational): Monitoring vs Activation Bursts in Regulated Agents

This paper clarifies an operational control boundary in regulated agents: awareness is the monitoring/gating layer, while thinking is the bounded activation burst that produces structured analysis artifacts.

Core focus:
- Separates awareness responsibilities (monitoring, posture selection, commit gating) from thinking responsibilities (candidate models, falsifiers, interventions).
- Defines a minimal awareness contract with inspectable inputs/outputs and reason codes.
- Defines a bounded thinking contract centered on structured artifact production rather than open-ended synthesis.
- Specifies transition and commit policy constraints for durable updates under uncertainty.
- Positions the distinction as governance/control architecture, not a theory-of-mind claim.
