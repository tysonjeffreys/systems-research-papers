## Summary
This note treats introspection as a measurement channel with state-dependent reliability. It defines epistemic load (a measurable proxy for compensation) and a corresponding reliability gate. It then specifies an admissibility policy: which classes of self-claims can update telemetry, plans, analysis artifacts, or durable commits.

Companion note: **Reflection Without Thrash** (RSML governor). RSML governance reduces epistemic load; admissibility rules constrain when self-claims can update durable state.
